# 🎯 Frontend Practice Repository

This repository contains my frontend development practice work, where I focused on learning and experimenting with individual HTML, CSS, and JavaScript concepts.

## 📚 What this repo includes

Instead of building full projects, I created small webpages to practice specific components, layouts, and interactions such as:

### 🎨 HTML & CSS Practice

* Box Shadow effects
* CSS Flexbox layouts
* CSS Grid basics
* Form elements and styling
* Positioning (absolute, relative, fixed, etc.)
* Colors, backgrounds, and gradients
* Typography and spacing

### ⚡ JavaScript Practice

* DOM manipulation (selecting and updating elements)
* Event handling (click, input, submit events)
* Basic form validation
* Simple interactive features (buttons, toggles, dynamic content)
* Beginner-level logic building using JavaScript

Each folder represents a separate concept or component that I practiced individually to strengthen my understanding step by step.

## 🛠️ Purpose

The goal of this repository is to:

* Build a strong foundation in frontend development
* Understand core concepts by practicing them in isolation
* Improve problem-solving skills using JavaScript
* Experiment with different UI patterns and behaviors

Each folder contains small, focused implementations of a specific concept.

## 🚀 Note

These are practice exercises and may not be fully responsive or production-ready.
The focus is on learning and experimentation rather than final design or optimization.

I will continue refining these concepts and applying them in more complete, real-world projects.
