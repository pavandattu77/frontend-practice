it's used to ass functionalities to a web page
First Brendon Eich developed a scripting language called echo later named as live script
To make it a open source they handed it to ECMA(European computer manufacturing Association) in year 1997 also called as ECMA script

Key features:
 1. It is a scripting language because each and every code written in JS can be read, analyse and execute
 2. It is a weekly typed language - if ; missing then also no error occurs
 3. It is dynamically typed language - if declare a variable and assign and store any type of data, later we can assign any value it might be string, Boolean
 4. It is synchronous single threaded/interpreted language 
 5. It's high level programming language
